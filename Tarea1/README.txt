# Pixelart
Para correr el código se tiene que usar el siguiente comando en la terminal:
python pixelart.py

Además se debe tener en la misma carpeta el archivo a interpretar que debe estar llamado codigo.txt

En caso de que exista un error el programa dará como salida un programa un archivo
errores.txt

En caso de que el programa haya funcionado correctamente se dará como salida un archivo
pixelart.png con la respectiva representación del codigo.txt

En caso de que exista errores de Runtime (mientras se ejecuta el problema) puede dar las siguientes respuestas:
    - No están las dos primeras lineas necesarias para la configuración del archivo
    - El color de fondo indicado no cumple con las condiciones necesarias (Probablemente el color no está correctamente definido)
    - El jugador se ha salido de la cuadricula o matríz 
    - El color indicado no cumple con las condiciones necesarias (Los numeros no están en el rango de valores de [0, 255])
De todas maneras en caso de errores de Runtime se indicará la linea del error para que pueda ser solucionada.

Si no hay información suficiente para el correcto funcionamiento del programa estoy a libre disposición de explicarlo!

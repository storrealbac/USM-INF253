# JavaQuest
Para poder ejecutar el codigo primero hay que compilarlo usando el siguiente comando
> make

Luego de haber compilado, debe crear la carpeta build; Despues de tener la carpeta build
se puede correr el programa de dos maneras
    > make runjava
    > java -jar JavaQuest.jar

(Al final, son el mismo comando, pero uno es mas corto C;)

Luego de eso puedes jugar tu aventura con Kiwi!
